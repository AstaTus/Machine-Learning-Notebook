## 数学

### 微积分

#### 求导公式：
​	$\frac{df(x)}{dx} = \lim_{\Delta x \rightarrow 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$



#### 链式法则

$\frac{\delta y}{\delta x} = \frac{\delta y}{\delta z}\frac{\delta z}{\delta x}$

标量的链式法则右侧是乘积，但是矩阵的链式法则，右侧是点积



### 对数

#### 对数四则运算

* $log(AB) = logA + logB$

* $log(\frac{A}{B}) = logA - logB$

* $logN^x = xlogN$


###导数
####导数四则运算
* $(u + v)'=u' + v'$

* $(u - v)' = u' - v'$

* $(uv)' = u'v + uv'$

* $(u/v)' = (u'v - uv')/v^2$
