## 数学

### 微积分

ML在微积分中主要用到的是求导

#### 求导公式：
​	$\frac{df(x)}{dx} = \lim_{\Delta x \rightarrow 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$



### 对数

#### 对数四则运算

* $log(AB) = logA + logB$

* $log(\frac{A}{B}) = logA - logB$

* $logN^x = xlogN$


###导数
####导数四则运算
* $(u + v)'=u' + v'$

* $(u - v)' = u' - v'$

* $(uv)' = u'v + uv'$

* $(u/v)' = (u'v - uv')/v^2$
